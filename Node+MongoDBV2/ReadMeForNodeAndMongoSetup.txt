(1) Download and clone Preston-Medical-Pharmacy
(2) make a folder in C Drive called node 
(3) Go to the command prompt and go to C:/node/
(4) Type npm install -g express-generator
(5) copy nodetest1 folder in node folder 
(6) Go to the command prompt and go to C:/node/nodetest1
(7) type npm install
(8) type npm start
(9) Go to chrome and type http://localhost:3000/helloworld in the address bar 
(10) Install mongodb from: https://www.mongodb.com/download-center?jmp=nav#community
(11) Open a new cmd prompt 
(12) Navigate to: C:\Program Files\MongoDB\Server\3.2\bin
(13) Create a folder data in C:/node/nodetest1/data
(14) Run mongod --dbpath c:\node\nodetest1\data\
(15) Run the server by typing mongo in cmd prompt
